Society Management System Requirements
User Roles
1. Admin
Login using hardcoded username and password stored in the database.
Total 3 Admin accounts.
Can perform Create, Read, Update, Delete (CRUD) operations on:
Members
Workers
Worker Attendance
Expenses
Maintenance Records
Can view Dashboard.
2. Society Member
Register and Login using the platform.
Can only view:
Dashboard
Members List
Workers List
Worker Attendance
Expenses
Maintenance Records
No Create, Update, or Delete permissions.
3. Worker
No login required.
Managed only by Admin.
Worker information is stored in the system and used for attendance management.
